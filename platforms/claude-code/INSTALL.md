# Claude Code — Install Guide

## Quick Install

```bash
bash scripts/install.sh claude-code
```

## MCP Setup (Optional)

```bash
cd scripts/
python3 setup.py
```

The script auto-creates `.mcp.json`. Restart Claude Code.

## Usage

Ask Claude:
- "Analyze my Google Ads campaigns from the last 30 days"
- "What's my Quality Score distribution?"
- "Review search terms and suggest negative keywords"
- "Deep dive into my Performance Max campaign"
- "Pause campaign X"
- "Add these negative keywords: [list]"
